## Program name: Simple Logic Algebra Calculator(demo)
## 项目名称: 简易逻辑代数运算器(试用版)

 - Program target users: Beginners of DLC(Digital Logic Circuit)  
 - 目标受众: 数字逻辑电路初学者
 <br><br>
 - Program function: To automatically simplify logic algebra functions(up to 5 variables)
 - 项目功能: 自动化简逻辑代数函数(最多5个逻辑变量)
 <br><br>
 - Quick start: Click **Releases** and install the version that suits your operation system
 - 快速开始: 点击**Releases**，找到适合您的操作系统的版本并下载
 <br><br>
 - Program requirements: You can view them in *requirements.txt*
 - 项目依赖: 见*requirements.txt*
 If you want to install the original code, run below in your terminal(virtual environment is preferred):
 如果您想获取源码，在您的终端中运行(建议在虚拟环境下操作):
```
git clone https://github.com/AncientMoon114/logicAlgebra.git
cd logicalAlgebra
pip install -r requirements.txt
python app.py
``` 
 <br><br>
 - How to use: Click **logicAlgebra** twice
 - 如何使用: 双击**logicAlgebra**
 <br><br>

## Several Things to Say
## 开发者的话
 - I'm just a university student. This is the first program I submit to github. The reason why I started this program was that I needed to learn Digital Logic Circuit. When I was doing experiments with chips, I found simplifying logic algebra functions by drawing Karnaugh maps consumes my time a lot. As a result, I decided to make it automatically, and the program is born. I have to admit that current version is far too rough, because I made it during my busy preparation for the final-terminal test. Maybe some days later I will make it better by adding functions such as solving driving equation in sequential logic circuit. I would appreciate it if someone can redevelop this program, since this is nothing, only able to simplify logic functions with at most five variables. 
 - 我只是一个大学生，这是我第一个上传到Github的项目。我做这个项目仅仅是出于我要学《数字逻辑电路》这门课。上实验课的时候要自己设计电路，然后简化逻辑表达式是一个很折磨的事情。然后我就想着能不能让电脑来帮我干，于是这个项目就诞生了。不得不承认这个项目做的很简陋，因为我拖到期末考复习的时候才完工。也许过几天我会完善一下功能，尤其是加入计算时序逻辑电路中的驱动方程。如果有大佬打算二次开发这个项目的话我直接跪了，毕竟这就是一个能简化五个变量以下逻辑表达式的小玩意，没什么太大的价值。
