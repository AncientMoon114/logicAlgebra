输入格式说明

支持两种格式，自动识别：

📝 简写格式（推荐）

ab变量并列 = 逻辑与（AND），如 ab = a ∧ b
a+b+ = 逻辑或（OR），如 a+b = a ∨ b
a'后缀 ' = 逻辑非（NOT），如 a' = ¬a
(x)'括号 + 后缀 '，如 (ab)' = ¬(a ∧ b)
( )括号决定优先级

🔧 方括号格式（兼容旧版）

[and]逻辑与（AND）
[or]逻辑或（OR）
[not]逻辑非（NOT）
[xor]异或（XOR）
[xnor]同或（XNOR）
( )括号决定运算优先级
a, b, c …单个字母表示变量（最多 5 个）

⚠ 简写优先级：'（最高）> 并列(AND) > +（最低）。多层嵌套请用括号明确。

📝 简写示例：(a+b)'c  →  ¬(a ∨ b) ∧ c
          ab'+c  →  (a ∧ ¬b) ∨ c
          ab+cd  →  (a ∧ b) ∨ (c ∧ d)
📝 方括号示例：(a[or]b)[and]([not]c)  →  (a ∨ b) ∧ ¬c
